# Website bán phụ kiện điện thoại

## Giới thiệu
Đây là website bán phụ kiện điện thoại được xây dựng để thực hành lập trình web.

## Công nghệ sử dụng
- Python (Flask)
- HTML
- CSS
- JavaScript
- SQLite

## Chức năng chính
- Hiển thị danh sách sản phẩm
- Xem chi tiết sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Quản lý dữ liệu sản phẩm

## Mục đích
Project được thực hiện nhằm luyện tập kỹ năng lập trình web và xây dựng ứng dụng web cơ bản.
